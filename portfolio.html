Furry Friends Finder
 Github repo

This full-stack Node.js application finds three top mathing small breeds of dogs based on a user's answers to five survey questions. The app displays the breeds and the images of these best matching small breeds. The app uses Express to handle routing and utilizes Css and Bootstrap for styling, and provides API routes to display small breeds of dogs.

what2grow
 Github repo

This is a web application designed by a small team to help amateur gardeners make informed decisions about best time to grow plans. The app provides the first suitable month and the optimal month for the plant to grow based on the user's geolocation, as well on the historical daily min, max, and mean temperatures for 2015-2018 years provided by aWhere API.

Crystals Collector Game
 Github repo

The web app is a guessing game when a user have to match the number randomly generated by a computer by pressing on the crystals images. All the crystals will have four randomly generated hidden values from 1 to 12, and the sum of the values of the crystals should exactly match the number generated by the computer.

Myazaki Moments
 Github repo

The web app uses Giphy API to retrieve images of Miyazaki movies, which names are displayed on the buttons generated by the app. When a user clicks on each gif, the source of the image is switched to the animated version of the gif, and that gives the impression that the image moves on a mouse click. The app uses Ajax, jQuery, Bootstrap 4, CSS, HTML and javascript.

Google Books
 Github repo
