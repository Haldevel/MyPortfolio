Furry Friends Finder
 Github repo

This full-stack Node.js application identifies three small dog breeds based on a user's survey answers. The app displays the breeds names and the images of the best matching small dog breeds. The application uses Node.js and Express to handle routing, utilizes CSS and Bootstrap 4 for styling, and provides API routes to display best matching breeds of dogs.

Trading Post
 Github back-end ☆ Github front-end

This is a decoupled MERN web app with MVC architecture pattern, designed to help people trade items without using money. It's a place where you can post and find items for exchange, and contact other users to make a trade with them. The API on the back-end utilizes Node.js, Express, MongoDB/Mongoose, and uses JWT for authentication. The front-end uses React, Redux, and Materialize.

what2grow
 Github repo

This is a web application designed by a small team to help amateur gardeners make informed decisions about the best time to grow plans. The app provides the first suitable month and the optimal month for the plant to grow based on the user's geolocation, as well on the historical daily min, max, and mean temperatures for 2015-2018 years provided by aWhere API.

Myazaki Moments
 Github repo

The web app uses the Giphy API to retrieve images of Miyazaki movies, with names that are displayed on the buttons generated by the app. When a user clicks on each gif, the source of the image is switched to the animated version of the gif, and that gives the impression that the image moves on a mouse click. The app uses Ajax, jQuery, Bootstrap 4, CSS, HTML and javascript.

Google Books
 Github repo

This web app is a product of collaboration between a small team. It is designed to help users to search for books using the Google books API, as well as save favorite books info. The books information can be saved (and deleted) in MongoDB. The app is a decoupled MERN app with the React front-end and the back-end utilizing Node.js, Express, and MongoDB. My contribution to the project included writing back-end API and routing, as well as some React components and the front-end routing.

Crystals Collector Game
 Github repo

The web app is a fun guessing game where users have to guess how many gems fit into a number randomly generated by a computer by pressing on the images of crystals. All the crystals will have four randomly generated hidden values from 1 to 12, and the sum of the values of the crystals should exactly match the number generated by the computer. The app uses javascript, jQuery, Bootstrap 4, CSS, and HTML.
