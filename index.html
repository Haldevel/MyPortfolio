About me

I am an experienced full-stack software developer living in the Bay Area with a demonstrated history of working on multiple front- and back-end projects for more than 9 years. I also have a Masters degree with honors in Electrical Engineering. For backend programming, I prefer using Node.js, but I also can use C#. I enjoy writing REST APIs using Node.js and Express and like to build front-end with React and Redux, utilizing Bootstrap and Materialize libraries for CSS/js support. I can use NoSQL (MongoDB, Firebase) and relational (MySQL, SQL Server) databases. I am passionate about any industry or field that utilizes modern web technologies. In the past, I developed and supported custom ETL pipelines to process pending XML files from multiple insurance carriers using Java, as well as developed web applications with dashboard features using MS SQL Server, C#, Node.js, javascript, jQuery, HTML, CSS, and Bootstrap. Some of my projects include building an automated email platform to manage all company daily email communications. I also developed multiple web clients to consume APIs from different vendors and carriers, and built some apps that provided APIs to function as communication channels for some medical and APS vendors to track case processes.
